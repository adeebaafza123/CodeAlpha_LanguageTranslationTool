# CodeAlpha Language Translation Tool

This project is an AI-based Language Translation Tool developed using Python and Tkinter.

## Features
- User-friendly GUI
- Source and target language selection
- Google Translation API integration
- Real-time text translation
- Copy translated text feature

## Technologies Used
- Python
- Tkinter
- deep-translator
- pyperclip

## How to Run
1. Install required libraries
2. Run translator.py
